## Setting Up the Python ENV
```bash
# Upgrade the PIP
python3 -m pip install --upgrade pip
# Setup the `venv'
python3 -m venv venv
# Activate the 'env'
source venv/bin/activate
```

## Installing the Sphinx
```bash
pip install sphinx
```
